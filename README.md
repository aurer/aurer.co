# Aurer

This repo contains the whole <aurer.co> website sans-content.

## Requirements

 * Local development server running PHP 7.1+ (e.g. Apache, nginx, MAMP)
 * Node.js 8+ + NPM

## Installation

### Development

* git clone git@github.com/aurer/aurer.co.git aurer.co/public
* cd aurer.co
* composer install
* yarn install
* yarn start

### Production

* git clone git@github.com/aurer/aurer.co.git aurer.co/public
* cd aurer.co
* composer install
